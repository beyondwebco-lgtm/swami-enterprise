# Swami Enterprise Developer Instructions

Inherits guidelines from `@AGENTS.md`.

## Quick Commands
- `npm run dev`: Start local development server on http://localhost:3000
- `npm run build`: Build production application & validate TypeScript types
- `npm run lint`: Run ESLint check

## Development Notes
- Color palette: `navy-dark`, `navy-slate`, `gold`, `gold-light`, `warm-white`.
- Media assets must be placed inside `public/`. Keep root directory clean of orphan images.
- Main page composition resides in `src/app/page.tsx`.
